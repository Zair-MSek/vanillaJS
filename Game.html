Vanilla JS: Tic-Tac-Toe

Play a fun game of Tic-Tac-Toe, built using HTML, CSS and plain JavaScript.

If this is your first time playing Tic-Tac-Toe, then you're in for much fun. To win the game, a player must place three of their marks in a horizontal, vertical, or diagonal row. For more info see Tic-Tac-Toe wiki's page.

Before you start

  • You need to enter names for the first and second player to activate the game.
  • Once activated, instructions are shown to indicate which player needs to make a move.
  • To make a move, click on an empty cell. Please note that moves are final and cannot be reversed.
  • Cells with value "X" represents moves the first player made.
  • Cells with value "O" represents moves the second player made.
